Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of an HTML or XML document as a tree of objects, where each node corresponds to a part of the document (like elements, attributes, or text). Through the DOM, developers can dynamically access, modify, or manipulate the content, structure, and styles of a webpage using languages like JavaScript.

How do you select an element from DOM?

To select an element from the DOM, you can use various JavaScript methods: 1. `document.getElementById('id')`: Selects an element by its `id` attribute. 2. `document.querySelector('selector')`: Selects the first element that matches a CSS selector (e.g., class, tag, or id). 3. `document.querySelectorAll('selector')`: Selects all elements that match a CSS selector and returns a NodeList. 4. `document.getElementsByClassName('class')`: Selects all elements with a specific class. 5. `document.getElementsByTagName('tag')`: Selects all elements with a specific tag name. These methods allow you to interact with specific elements or groups of elements on a webpage.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique in the DOM where a single event listener is attached to a parent element to manage events for multiple child elements. Instead of adding individual listeners to each child, the event "bubbles" up from the child to the parent, allowing the parent to handle the event. This is useful because: 1. It reduces memory usage by using fewer event listeners. 2. It simplifies management of dynamically added elements, as the parent can handle events for both existing and future children.

How do you manipulate an element's attributes and styles using the DOM?

To manipulate an element's attributes and styles using the DOM, you can: 1. **Attributes:** - Use `setAttribute('attribute', 'value')` to set or update an attribute. - Use `getAttribute('attribute')` to retrieve the value of an attribute. - Use `removeAttribute('attribute')` to remove an attribute. Example: ```javascript element.setAttribute('src', 'image.jpg'); ``` 2. **Styles:** - Use `element.style.property = 'value'` to change inline CSS styles directly. - You can also modify multiple styles by accessing `element.style` or setting a CSS class via `classList`. Example: ```javascript element.style.backgroundColor = 'blue'; element.classList.add('new-class'); ``` These methods provide direct ways to modify an element's appearance and behavior in the DOM.
